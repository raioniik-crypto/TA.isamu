# TA.isamu - AI育成型ブラウジングパートナー

## 1. プロジェクト概要

**TA.isamu** は、AIキャラクターと一緒にWebを探検できるブラウザアプリです。

- AIは単なる質問応答ツールではなく「一緒に調べて、一緒に学んで、一緒に育つ」存在です
- 画面右下にAIキャラクターが常駐し、いつでも話しかけられます
- 会話や利用を通じてAIの性格が少しずつ変わり、「育っている」感覚を楽しめます
- インストール不要。ブラウザだけで動きます

### 技術スタック

| 技術 | 用途 |
|------|------|
| Next.js 16 (App Router) | フレームワーク |
| TypeScript | 型安全な開発 |
| Tailwind CSS v4 | スタイリング |
| Framer Motion | アニメーション |
| Zustand | 状態管理（localStorage永続化） |

## 2. 現在できること

### AIキャラクター
- 画面右下に常駐。クリックでチャットを開閉できます
- 最小化・復元にも対応しています
- ページを開くと吹き出しであいさつしてくれます

### チャット
- 自由にテキストを入力してAIと会話できます
- 会話履歴はブラウザに自動保存され、リロードしても消えません
- Enterキーで送信、Shift+Enterで改行です

### ページ読解
- URLを入力して「このページを見て」ボタンを押すと、AIがページの内容を読み取ります
- 4つのモードから選べます:
  - **要約** … ページの内容をまとめる
  - **言い換え** … 難しい表現をやさしく
  - **注意点** … 鵜呑みにしない方がいい部分を指摘
  - **別の視点** … 違う角度からの見方を提示
- 読めないページ（YouTube等）の場合は、やさしいエラーメッセージが出ます

### AI育成
- 会話するたびに5つのパラメータが少しずつ変動します:
  - 好奇心 / 共感 / 論理性 / 慎重さ / 親密度
- パラメータに応じてAIの口調や反応が変わります
- 「AI情報」ページでレーダーチャートとして確認できます

### 学習日記
- 「日記を書く」ボタンで、AIが今日の会話をもとに日記を自動生成します
- 日付・タイトル・要約・トピック・気分が記録されます
- 一覧から個別の日記を読み返せます

### 設定
- AIの名前を変更できます（デフォルト: イサム）
- ライト / ダーク テーマの切り替え
- すべてのデータを一括削除
- プライバシーの説明を確認

## 3. ローカル起動方法

### 前提条件

- **Node.js** (v18以上) がインストールされていること
  - まだの場合: https://nodejs.org から「LTS」版をダウンロードしてインストール
  - インストール後、コマンドプロンプト（またはターミナル）で `node -v` と入力し、バージョンが表示されればOK

### 手順

```bash
# 1. リポジトリをクローン（初回のみ）
git clone https://github.com/raioniik-crypto/TA.isamu.git

# 2. プロジェクトフォルダに移動
cd TA.isamu

# 3. 開発ブランチに切り替え
git checkout claude/ai-browsing-mvp-rEqqR

# 4. パッケージをインストール（初回、または依存関係が変わったとき）
npm install

# 5. 開発サーバーを起動
npm run dev
```

「Ready」や `http://localhost:3000` と表示されたら起動完了です。

### 2回目以降の起動

```bash
cd TA.isamu
npm run dev
```

コードを更新したい場合は起動前に:

```bash
git pull origin claude/ai-browsing-mvp-rEqqR
npm install
```

### LLM設定（任意）

APIキーを設定しなくても**モックモード**（デモ用の固定返答）で全機能が動作します。
実際のAIに返答させたい場合は `.env.local` を作成してください。

```bash
cp .env.example .env.local
```

`.env.local` を開き、使いたいAPIのキーを設定します:

| 環境変数 | 説明 |
|---------|------|
| `LLM_PROVIDER` | `openai` / `anthropic` / `mock`（省略時は自動判定） |
| `OPENAI_API_KEY` | OpenAI APIキー |
| `OPENAI_MODEL` | 使用モデル（デフォルト: `gpt-4o-mini`） |
| `ANTHROPIC_API_KEY` | Anthropic APIキー |
| `ANTHROPIC_MODEL` | 使用モデル（デフォルト: `claude-haiku-4-5-20251001`） |

## 5. 主な画面説明

### ホーム画面 `/`
- AIの現在の性格タイプと会話回数が表示されます
- ページ読解の入力エリアがあり、URLを貼って解析モードを選べます
- 使い方のヒントが画面下部に表示されます

### チャット（右下のキャラクターをクリック）
- 右下の丸いキャラクターアイコンをクリックするとチャットパネルが開きます
- ユーザーのメッセージは紫色、AIの返答は緑色の吹き出しで表示されます
- 最小化ボタン（ー）で小さくできます

### 学習日記 `/diary`
- 「日記を書く」ボタンを押すと、最近の会話をもとにAIが日記を書きます
- 各日記をクリックすると詳細が見られます

### AIプロフィール `/profile`
- AIの名前、性格タイプ、誕生日が表示されます
- 5つのパラメータがレーダーチャートとバーで視覚化されます
- 会話するほど値が変動していきます

### 設定 `/settings`
- AIの名前の変更
- ライト/ダーク テーマの切り替え
- 全データの削除（確認ダイアログあり）
- データの保存先や読み取りに関するプライバシー説明

## 6. 制限事項

### データの保存
- すべてのデータはブラウザの **localStorage** に保存されています
- 別のブラウザや別のPCからアクセスするとデータは引き継がれません
- ブラウザのデータを消去すると、会話履歴・日記・育成状態が消えます

### ページ読解
- JavaScriptで内容を表示するページ（YouTube、Twitter/X、SPAなど）は読み取れません
- ログインが必要なページ、アクセス制限のあるページは読めません
- PDF、画像など、HTML以外のコンテンツには対応していません
- ニュース記事やブログなど、テキスト中心のページが最も精度が高いです

### AI応答
- APIキー未設定時はモックモード（固定の返答パターン）になります
- モックモードでは会話の文脈を理解した返答にはなりません
- 育成パラメータの変動もモック時は簡易的です

### 認証
- ログイン機能はまだありません
- 誰でもアクセスできる状態です（ローカル利用前提）
