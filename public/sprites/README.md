# Sprite Assets

ここにフィルのスプライト画像を配置します。

## 命名規則

```
phil-{state}-{frame}.png
```

### 状態一覧

| state | 用途 | 想定フレーム数 |
|-------|------|--------------|
| idle | 待機 | 2-4 |
| blink | 瞬き | 3-5 |
| walk | 歩行 | 4-8 |
| surprised | 驚き | 2-4 |
| sit | 座り | 2-4 |

### 例

```
phil-idle-0.png
phil-idle-1.png
phil-blink-0.png
phil-blink-1.png
phil-blink-2.png
phil-walk-0.png
phil-walk-1.png
phil-walk-2.png
phil-walk-3.png
...
```
